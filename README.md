# trmnl-tado-rooms

TRMNL plugin / plugim per mostrar temperatures i humitat de les habitacions de Tado.

## Idea

Aquest repo prepara una sortida simple perquè TRMNL pugui renderitzar les habitacions i temperatures actuals.

## Dades actuals d'exemple

- Menjador: 19.63°C
- Cuina: 18.92°C
- Habitació papes: 20.15°C
- Habitació nenis: 18.32°C
- Lavabo: 18.91°C
- Estudi: 17.37°C

## Components previstos

- script que consulta Tado
- endpoint o fitxer JSON de sortida
- template/render per TRMNL
